## Como usar

Luego de descargar los archivos, hay que ejecutar el siguiente comando desde la carpeta principal.

```console
npm install
```
Esto instalará las dependencias del servidor.

Ahora solo queda iniciarlo con el siguiente comando.

```console
node app.js
```
